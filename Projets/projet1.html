Diplomacy

Site web de partage d'astuce pour un jeux vidéo. Les utilisateurs peuvent poster des articles avec images et de laisser des commentaires sur ceux des autres.

Présentation

Technologies utilisées: React, Api, NodeJs, HTML, CSS, JavaScript, MongoDB

Le site est réalisé avec React en JavaScript.
Les articles et les commentaires sont stockés sur une base de donnée MongoDB. Ils sont accessibles et modifiables via une API realisé avec node JS.

Le site comporte un systeme de LOGIN. Seul les utilisateurs enregistré peuvent ajouter / modifier ou supprimer des articles. En revanche tout les utilisateurs peuvent laisser un commentaire.

Le site est hebergé sur un serveur VPS unbuntu.

L'API est disponible à l'adresse : http://test3.pascalloria.fr
