Le seigneur des pendus

Un jeu du pendu, dans l'univers de TOLKIEN. Trouvez le mot avant de finir pendu !

Présentation

Technologies utilisées: HTML, CSS, JavaScript, SVG

Le jeux choisi un mot parmis un dictionnaire prédéfini , vérifie que ce mot n'est pas celui qui à été joué le coups d'avant. Les lettres du mots sont ensuite remplacer par "_" et afficher à l'utilisateur. Celui dispose d'un Input permettant de proposer une lettre ou un mot, et d'un clavier virtuel permettant de proposer une lettre en clickant sur celle ci. Lorsqu'une lettre est proposé le bouton correspondant est désactivé. Le programme vérifie la présence de la lettre dans le mot. En cas de présence la lettre est revelé dans le mot coder. Dans le cas contraire un trait du pendu est déssiné.

Ici le but est de faire deviner des personnages de l'univers de TOLKIEN. Pour aider l'utilisateur la race du personnage est affiché. En fin de partie un son et un message sont joué et affiché selon que la partie soit gagné ou perdu. Et dans tous les cas un lien vers la page Wikipédia du personnage est proposé.

Le site est hebergé par gitHub avec page.
